## Sobre
Desenvolvimento de APIs REST utilizando Ktor e linguagem Kotlin.

## Pré-requisitos
* JDK 17
* Maven 3.8.x
* IntelliJ
* MySQL 8
* Postman

## Projeto
O projeto desenvolvido no curso será uma API para gerenciamento de despesas pessoais.
